# 🎓 Student Performance Predictor

An ML-powered web application that predicts student GPA based on academic and behavioral factors.

## Features
- Predict final GPA using Random Forest Regression
- Interactive Streamlit dashboard
- Real-time performance analysis
- 300 synthetic student records for training

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Installation

```bash
pip install -r requirements.txt
python train.py
python -m streamlit run app.py
